# Contributing

1. Open an issue describing the problem or extension.
2. Create a focused branch.
3. Run `bun run hooks:install` once after cloning.
4. Keep migrations backward compatible and property-scope every query.
5. Run `bun run gate` before submitting a pull request or publishing a branch.
6. Include screenshots for UI changes and explain any security-sensitive behavior.

The local gate and repository Git hooks are the source of truth. Contributions must not rely on GitHub Actions being enabled or available.
